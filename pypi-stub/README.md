# 1-800-operator

This package is a **placeholder**. `operator` is not installable via pip because it ships
with a browser, a persistent Chrome profile, and other components that pip cannot
manage cleanly.

## Install

```bash
curl -sSf https://1-800-operator.com/install | sh
```

## Why this placeholder exists

Reserved on PyPI to prevent the name from being claimed by an unrelated project that
could confuse users who search PyPI for `operator`.

The actual `operator` source lives at https://github.com/1-800-operator
